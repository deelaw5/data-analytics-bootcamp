# Data Analytics Bootcamp

Welcome to the Data Analytics Bootcamp! This repository contains all the materials, assignments, and resources for learning data analytics from the ground up.

## Overview

This bootcamp is designed to take you through a comprehensive journey in data analytics, covering everything from database design and SQL queries to data visualization and analysis techniques. Each week builds upon the previous one, providing you with hands-on experience and practical skills.

## Weekly Content

- [Week 1](./week%201/) - Introduction to Databases and ERD
- [Week 2](./Week%202/) - Joining Data in SQL

## How to Use This Repository

### Getting Started

1. **Navigate to Weekly Folders**
   - Click on any week folder (e.g., "week 1") to view its contents
   - Each week has its own folder containing relevant materials
   - Start with Week 1 and progress sequentially
   - Each week folder contains:
     - Assignment instructions
     - Required resources (datasets, images, etc.)
     - Any additional materials needed for that week

2. **View and Download Materials**
   - Click on files to view them directly in GitHub
   - For PDFs and images, you can view them in the browser
   - Use the "Download" button to save files to your computer if needed

3. **Create Your Branch**
   - Click on the branch dropdown (usually shows "main" or "master")
   - Type your name in the "Find or create a branch" field (e.g., "john-doe")
   - Click "Create branch: [your-name] from 'main'"
   - This creates your personal branch where you'll upload all your work

4. **Upload Your Work**
   - Make sure you're on your personal branch (check the branch dropdown)
   - Navigate to the appropriate week folder
   - Click "Add file" â†’ "Upload files"
   - Upload your completed assignments
   - Add a commit message describing your work
   - Click "Commit changes"
   - Always upload your work to your personal branch, not the main branch

5. **Submit Your Work (Every Saturday)**
   - Every Saturday, create a Pull Request (PR) to submit your weekly work
   - Click on the "Pull requests" tab
   - Click "New pull request"
   - Set the base branch to "main" and compare branch to your personal branch
   - Add a title and description of your work for the week
   - Click "Create pull request"
   - Your work will be reviewed and merged

6. **Follow the Weekly Structure**
   - Read the assignment instructions for each week
   - Complete the exercises and assignments
   - Review the provided resources
   - Upload your completed work to your personal branch throughout the week
   - Submit a PR every Saturday with your weekly progress

### Best Practices

- **Stay Organized**: Keep your work organized by week
- **Practice Regularly**: Consistent practice is key to mastering data analytics
- **Ask Questions**: Don't hesitate to review materials multiple times
- **Track Progress**: Mark completed assignments and note any challenges you encounter
